# WantToSay

### 1. 서비스 소개

---

유튜브 플랫폼의 동영상과 인터넷 영문 기사들로 영어 스피킹 연습을 할 수 있는 안드로이드 어플리케이션 입니다. 

학교 팀 프로젝트로 총 3명의 팀원과 함께 진행하였습니다.
